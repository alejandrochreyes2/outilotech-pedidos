# outiltech-claude

Carpeta de trabajo para el proyecto **outiltech-claude**, basado en el proyecto existente `proyecto-pedidos`.

## Propósito

Este directorio contiene todos los nuevos desarrollos, modificaciones, documentación y entregables que se generen a partir de ahora, tomando como base lo ya construido en `proyecto-pedidos` (backend, frontend, configuración Docker, scripts Supabase, Terraform, etc.).

## Estructura base de referencia (proyecto-pedidos)

- `backend/` — API y lógica de servidor
- `frontend/` — Interfaz de usuario
- `config/` — Configuraciones del proyecto
- `scripts/` — Scripts de utilidad
- `terraform/` — Infraestructura como código
- `arquitectura/` — Diagramas y diseño
- `documentación del proyecto/` — Documentación funcional/técnica
- `docker-compose.yml`, `docker-compose.prod.yml`, `docker-compose.azure.yml` — Orquestación de contenedores
- `supabase_tables.sql`, `supabase_poblar_datos.sql`, `manual_init.sql` — Scripts de base de datos

## Convenciones de trabajo

- Todo lo nuevo se guarda dentro de `outiltech-claude/`.
- No modificar los archivos originales de `proyecto-pedidos/` salvo indicación expresa.
- Cuando se requiera tomar algo del proyecto base, se copia o se referencia desde aquí.
